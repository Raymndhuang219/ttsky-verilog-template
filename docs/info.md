## How it works

This project is a reaction time tester. The design uses a single button to start and react, and displays the result in milliseconds on three multiplexed 7-segment displays.

The design cycles through four states:

- **IDLE** — The display shows `---`, indicating the device is ready. Press the button to start.
- **WAITING** — The display goes blank while the design waits for a random delay of 1–5 seconds, generated by a 16-bit Galois LFSR. Pressing the button during this phase counts as an early press and resets back to IDLE.
- **GO** — The display shows `Go`, and the millisecond counter starts. Press the button as fast as possible.
- **RESULT** — The display shows your reaction time in milliseconds (0–999). Press the button again to reset and play another round.

The 8 output pins (`uo_out`) carry the 7-segment data for all three digits simultaneously. The three digit select pins (`uio_out[2:0]`) cycle through hundreds, tens, and units at ~1 kHz, which makes the display appear solid to the human eye (multiplexing).

The clock is assumed to be 50 MHz. The `CLK_HZ` parameter must match your actual clock frequency for accurate millisecond timing.

## How to test

1. Set the clock to **50 MHz**.
2. Wire a button between `ui_in[0]` and VCC (active high). The button input has a built-in synchroniser and edge detector.
3. Wire three common-cathode 7-segment displays:
   - Connect all segment pins (a–g, dp) in parallel across all three displays, driven by `uo_out[7:0]`.
   - Connect the common cathode of the **hundreds** digit to `uio[2]`.
   - Connect the common cathode of the **tens** digit to `uio[1]`.
   - Connect the common cathode of the **units** digit to `uio[0]`.
4. Apply reset (`rst_n` low, then high).
5. The display should show `---`. Press the button to start.
6. Wait for the display to show `Go`, then press the button as fast as you can.
7. Your reaction time in milliseconds will appear on the display.
8. Press the button again to reset and play another round.

**Early press:** If you press the button before `Go` appears, the design resets silently back to `---`.

## External hardware

- 3× 7-segment LED displays
- 1× momentary push button
- Current-limiting resistors for each segment
